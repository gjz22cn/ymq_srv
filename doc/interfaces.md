### 1.2 返回报文格式
```
{
  ret: , // 0-成功；非0-失败
  data: , //数据
  msg:  , //失败原因
}
```
### 1.3 返回值定义
|ret|说明|
|:----|:----:|
|0|成功|
|8001|没有数据|
|8002|没有登陆|
|8003|无权限|

### 1.4 返回报文示例
### 1.4.1 POST/PUT/DELETE成功回复报文
```
{
  ret: 0
}
```
### 1.4.2 GET成功回复报文
```
//单实例
{
  ret: 0,
  data: {
  }
}

//多实例
{
  ret: 0,
  data: [
    {
    },
    {
    }
  ]
}
```
### 1.4.3 失败回复报文
```
{
  ret: [非0值],
  msg: ""
}
```
